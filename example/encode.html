WAV to FLAC conversion

Convert a WAV-file into FLAC format.
Either use the drop zone (change the extension of the WAV-file before dropping it into the browser - or else the WAV-file might be played back by the browser instead of converted to the FLAC format) or the file selection button.
By checking the "Download" box you can select, whether the FLAC-file should be downloaded.
